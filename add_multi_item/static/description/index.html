Add Multi Item

Add multi item by select items once.A powerful tools to help you quickly insert items in order line.

Key features
  • Can be used for any business object.
  • Support two way to add multi items in order line.
  • Easly using the function just by add options="{'create_row':True}" or options="{'add_batch_out':True}" in form view to relation field of order line . Accounding to the demo views.
  • Option "{'add_batch_out':True}" will add a "Add Multi Item" button in order line.
  • Automatically locate the first row of a batch of added records,which is convenient for quickly edit the quantity and price of the order line.
  • Support odoo version 13 .
